### File Codes

#### 1. `app.py`
```python
from flask import Flask, render_template, request, redirect, url_for
import json
import os

app = Flask(__name__)

# File to store user data
DATA_FILE = 'user_data.json'

# Initialize user data file if it doesn't exist
if not os.path.exists(DATA_FILE):
    with open(DATA_FILE, 'w') as f:
        json.dump([], f)

@app.route('/')
def home():
    return render_template('index.html')

@app.route('/submit', methods=['POST'])
def submit():
    if request.method == 'POST':
        user_info = {
            'name': request.form['name'],
            'email': request.form['email'],
            'age': request.form['age']
        }
        
        # Load existing data
        with open(DATA_FILE, 'r') as f:
            data = json.load(f)

        # Append new user data
        data.append(user_info)

        # Save updated data back to file
        with open(DATA_FILE, 'w') as f:
            json.dump(data, f)

        return redirect(url_for('success'))

@app.route('/success')
def success():
    return render_template('success.html')

if __name__ == '__main__':
    app.run(debug=True)
```

# User Info Flask App

This is a simple Flask application that allows users to input their information and stores it locally in a JSON file.

## Project Structure

```
flask_user_info/
│
├── app.py             # Main application code
├── user_data.json     # Stores user information
├── templates/         # Contains HTML templates
│   ├── index.html     # Form for user input
│   └── success.html   # Success page after form submission
└── README.md          # Project documentation
```

## Requirements

- Python (>= 3.6)
- Flask

## Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd flask_user_info
   ```

2. Install Flask:
   ```bash
   pip install Flask
   ```

## Running the Application

1. Run the Flask application:
   ```bash
   python app.py
   ```

2. Open your browser and navigate to `http://127.0.0.1:5000/`.

## Features

- User input form with fields for name, email, and age.
- Data is stored locally in a JSON file.
- Success page displayed after form submission.
